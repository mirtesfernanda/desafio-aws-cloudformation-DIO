# 📝 Notas e Aprendizados

- O CloudFormation é uma ferramenta poderosa para **Infraestrutura como Código (IaC)**.  
- Permite automatizar e versionar o provisionamento de recursos AWS.  
- A integração nativa com serviços da AWS garante confiabilidade e segurança.  
- O uso de stacks facilita a organização e controle das implantações.  
- É ideal para desenvolvedores e engenheiros DevOps que desejam padronizar ambientes.
